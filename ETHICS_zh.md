# 负责任使用声明

## FicForge 是什么

FicForge 是一个本地运行的 AI 辅助写作工具，面向想要自己创作同人作品的写手。
它不是用来替代人类创作者的。

## 数据与隐私

- **所有数据保存在你自己的电脑上。** FicForge 不会向 FicForge 自己的服务器上传任何数据。当你使用 AI 写作功能时，你的内容会被发送到你自己配置的 API 提供商（如 DeepSeek）——这是 AI 生成所必需的，且完全由你控制。
- **你的创作内容不会被用于训练任何 AI 模型。** FicForge 调用你自己配置的 API（如 DeepSeek）。许多主流模型服务商在其 API 服务条款中表示，API 调用的数据不会被用于模型训练（这通常与网页版对话产品不同）。具体请以你配置的 API 提供商的最新隐私协议为准。FicForge 本身不收集、不存储、不传输你的数据。
- **内置的向量检索完全在本地运行。** 不经过任何云端服务。

## 内容来源

- **请只导入你自己创作或拥有合法使用权的内容。**
- FicForge 的导入功能用于导入你自己的已有作品（如从其他平台迁移你自己的连载），不应被用于导入他人的作品。
- 因为 FicForge 坚持纯本地运行和绝对的隐私保护，系统没有权限、也不会扫描或验证你本地设备上的文件来源。因此，确保内容来源合法的全部责任在于使用者本人。

## 发布与标注

- 如果你将 AI 辅助创作的内容发布到任何平台，**我们建议标注 AI 的参与。**
- 尊重你所在创作社区的规则。许多同人社区对 AI 辅助创作有明确的标注要求或限制，请遵守相关规定。
- FicForge 的导出功能提供可选的 AI 辅助标注，默认开启。

## 我们的立场

FicForge 为找不到喜欢的作品、想要自己动手创作的人提供工具支持。
我们尊重所有创作者——无论使用什么工具创作——的劳动和权利。
我们反对任何形式的抄袭、冒充原创，或未经授权使用他人作品的行为。

## 为什么选择开源

选择 AGPLv3 开源协议，是为了确保 FicForge 始终作为一个公开、透明且不可被闭源商用的工具存在。我们拒绝将 AI 技术黑盒化，也拒绝他人利用本项目的开源成果进行不透明的牟利。
