### Why AGPL-3.0?

This project is built for the fanfiction writing community. We want it to stay open and free for individual creators. AGPL-3.0 ensures that:

1. The code stays open — anyone can inspect, learn from, and improve it
2. No one can take this work, close-source it, and sell it without contributing back
3. The original maintainers retain the ability to offer commercial licensing for enterprise use cases
