### 你的下一步操作

```
1. 按 GITHUB_SETUP_GUIDE.md 设置仓库（15 分钟）
   → 添加 LICENSE、LICENSING.md、.gitignore、README.md
   → 把所有治理文档和 PRD 放进仓库
   → 首次 commit 推送

2. 开始第一个任务 T-001（项目骨架搭建）
   → git checkout -b claude/t001-project-scaffold
   → 用 TASK_TEMPLATE.md 写任务单
   → 启动 Claude Code，贴 AGENT_PROMPTS.md 通用段 + 任务单
   → 审核 → 合并

3. 遇到问题随时找我
   → PRD 模糊点
   → Git 操作卡住
   → AI 产出和 PRD 冲突
   → 需要补充 DECISIONS.md
```
